## Ασφάλεια

Η Microsoft λαμβάνει σοβαρά υπόψη την ασφάλεια των προϊόντων λογισμικού και υπηρεσιών μας, που
περιλαμβάνει όλα τα αποθετήρια πηγαίου κώδικα στις οργανώσεις μας στο GitHub.

**Παρακαλούμε μην αναφέρετε ευπάθειες ασφαλείας μέσω δημόσιων θεμάτων στο GitHub.**

Για πληροφορίες αναφοράς ασφαλείας, τοποθεσίες, στοιχεία επικοινωνίας και πολιτικές,
παρακαλούμε ανατρέξτε στις τελευταίες οδηγίες για τα αποθετήρια της Microsoft στη
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
