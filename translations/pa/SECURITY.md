## ਸੁਰੱਖਿਆ

ਮਾਈਕ੍ਰੋਸਾਫਟ ਸਾਡੇ ਸਾਫਟਵੇਅਰ ਉਤਪਾਦਾਂ ਅਤੇ ਸੇਵਾਵਾਂ ਦੀ ਸੁਰੱਖਿਆ ਨੂੰ ਗੰਭੀਰਤਾ ਨਾਲ ਲੈਂਦਾ ਹੈ, ਜਿਸ ਵਿੱਚ ਸਾਡੇ GitHub ਸੰਗਠਨਾਂ ਵਿੱਚ ਸਾਰੇ ਸੋਰਸ ਕੋਡ ਰਿਪੋਜ਼ਟਰੀਜ਼ ਸ਼ਾਮਲ ਹਨ।

**ਕਿਰਪਾ ਕਰਕੇ ਸੁਰੱਖਿਆ ਦੀਆਂ ਕਮਜ਼ੋਰੀਆਂ ਨੂੰ ਜਨਤਕ GitHub ਮੁੱਦਿਆਂ ਰਾਹੀਂ ਰਿਪੋਰਟ ਨਾ ਕਰੋ।**

ਸੁਰੱਖਿਆ ਰਿਪੋਰਟਿੰਗ ਜਾਣਕਾਰੀ, ਸਥਾਨ, ਸੰਪਰਕ ਜਾਣਕਾਰੀ, ਅਤੇ ਨੀਤੀਆਂ ਲਈ, ਕਿਰਪਾ ਕਰਕੇ ਮਾਈਕ੍ਰੋਸਾਫਟ ਰਿਪੋਜ਼ਟਰੀਜ਼ ਲਈ ਤਾਜ਼ਾ ਮਾਰਗਦਰਸ਼ਨ ਨੂੰ ਵੇਖੋ [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)।
