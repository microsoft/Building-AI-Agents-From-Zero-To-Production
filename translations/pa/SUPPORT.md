# Support

## How to file issues and get help  

ਇਹ ਪ੍ਰੋਜੈਕਟ ਬੱਗ ਅਤੇ ਫੀਚਰ ਬੇਨਤੀਆਂ ਨੂੰ ਟਰੈਕ ਕਰਨ ਲਈ GitHub Issues ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ। ਕਿਰਪਾ ਕਰਕੇ ਨਕਲਾਂ ਤੋਂ ਬਚਣ ਲਈ ਨਵੇਂ ਮੁੱਦੇ ਦਰਜ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਮੌਜੂਦਾ ਮੁੱਦਿਆਂ ਦੀ ਖੋਜ ਕਰੋ। ਨਵੇਂ ਮੁੱਦਿਆਂ ਲਈ, ਆਪਣਾ ਬੱਗ ਜਾਂ ਫੀਚਰ ਬੇਨਤੀ ਇੱਕ ਨਵੇਂ Issue ਵਜੋਂ ਦਰਜ ਕਰੋ।

ਇਸ ਪ੍ਰੋਜੈਕਟ ਦੀ ਵਰਤੋਂ ਬਾਰੇ ਸਹਾਇਤਾ ਅਤੇ ਸਵਾਲਾਂ ਲਈ, ਕਿਰਪਾ ਕਰਕੇ **REPO MAINTAINER: ਇੱਥੇ ਦਿਓ ਨਿਰਦੇਸ਼ ਕਿ ਕਿਵੇਂ ਰਿਪੋ ਮਾਲਕਾਂ ਜਾਂ ਕਮਿਊਨਿਟੀ ਨਾਲ ਸਹਾਇਤਾ ਲਈ ਸੰਪਰਕ ਕਰਨਾ ਹੈ। ਇਹ ਕੋਈ Stack Overflow ਟੈਗ ਜਾਂ ਹੋਰ ਚੈਨਲ ਹੋ ਸਕਦਾ ਹੈ। ਤੁਸੀਂ ਲੋਕਾਂ ਦੀ ਕਿੱਥੇ ਮਦਦ ਕਰੋਗੇ?**।
