## Security

Microsoft သည် ကျွန်ုပ်တို့၏ ဆော့ဖ်ဝဲ ထုတ်ကုန်များနှင့် ဝန်ဆောင်မှုများ၏ လုံခြုံရေးကို အလေးထားပါသည်၊ ၎င်းတွင် ကျွန်ုပ်တို့၏ GitHub အဖွဲ့အစည်းများရှိ မူရင်းကုဒ်သိုလှောင်ရာများအားလုံး ပါဝင်သည်။

**လုံခြုံရေးအားနည်းချက်များကို အများပြည်သူ GitHub ပြဿနာများမှတဆင့် မသတင်းပို့ပါနှင့်။**

လုံခြုံရေးသတင်းပို့ခြင်းဆိုင်ရာ အချက်အလက်များ၊ တည်နေရာများ၊ ဆက်သွယ်ရန်အချက်အလက်များနှင့် မူဝါဒများအတွက် Microsoft သိုလှောင်ရာများအတွက် နောက်ဆုံးလမ်းညွှန်ချက်ကို
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md) တွင် ပြန်လည်သုံးသပ်ပါ။
