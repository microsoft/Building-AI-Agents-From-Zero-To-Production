## 🌱 စတင်ခြင်း

ဤသင်တန်းတွင် AI Agent များ တည်ဆောက်ခြင်းနှင့် ထုတ်လုပ်ခြင်း၏ အခြေခံအချက်များကို သင်ကြားပေးပါသည်။

သင်ခန်းစာတိုင်းသည် ယခင်သင်ခန်းစာပေါ်တွင် အခြေခံထားသည်ဖြစ်သောကြောင့် အစမှ စတင်၍ အဆုံးအထိ တက်ကြွစွာ လေ့လာရန် အကြံပြုပါသည်။

AI Agent ခေါင်းစဉ်များကို ပိုမိုလေ့လာလိုပါက [AI Agents For Beginners Course](https://aka.ms/ai-agents-beginners) ကို ကြည့်ရှုနိုင်ပါသည်။

### အခြားသင်ယူသူများနှင့် တွေ့ဆုံခြင်း၊ မေးခွန်းများ ဖြေကြားခြင်း

AI Agent များ တည်ဆောက်ရာတွင် အခက်အခဲရှိပါက သို့မဟုတ် မေးခွန်းများရှိပါက [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) တွင် ကျွန်ုပ်တို့၏ သီးသန့် Discord ချန်နယ်သို့ ဝင်ရောက်ဆက်သွယ်နိုင်ပါသည်။

### လိုအပ်သောအရာများ

သင်ခန်းစာတိုင်းတွင် ကိုယ်ပိုင် ကုဒ်နမူနာများ ပါရှိပြီး ဒေသတွင်းတွင် လည်ပတ်နိုင်ပါသည်။ ကိုယ်ပိုင် မိတ္တူတစ်ခု ဖန်တီးရန် [ဤ repo ကို fork လုပ်နိုင်ပါသည်](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)။

ဤသင်တန်းတွင် လက်ရှိ အသုံးပြုနေသော အရာများမှာ -

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

စတင်မတိုင်မီ ဤဝန်ဆောင်မှုများကို အသုံးပြုခွင့် ရှိကြောင်း သေချာစေပါ။

မော်ဒယ် ဟိုစတင်းနှင့် ဝန်ဆောင်မှုများဆိုင်ရာ နောက်ထပ် ရွေးချယ်စရာများ မကြာမီ ထပ်မံဖြစ်လာမည်ဖြစ်သည်။

## အထောက်အပံ့ ပေးခြင်း

ဤပရောဂျက်သည် အထောက်အပံ့များနှင့် အကြံပြုချက်များကို ကြိုဆိုပါသည်။ အများဆုံး အထောက်အပံ့များသည် သင်၏ အခွင့်အရေးရှိကြောင်းနှင့် အမှန်တကယ် သုံးစွဲခွင့်ပေးသည်ကို ကြေညာသည့် Contributor License Agreement (CLA) သို့ သဘောတူရမည်။ အသေးစိတ်အချက်အလက်များအတွက် <https://cla.opensource.microsoft.com> ကို ကြည့်ရှုပါ။

သင် pull request တင်သည့်အခါ CLA bot သည် သင် CLA ပေးရန် လိုအပ်မရှိမရှိကို အလိုအလျောက် သတ်မှတ်ပြီး PR ကို သင့်တော်စွာ အလှဆင်ပေးပါမည် (ဥပမာ - status check, comment)။ bot မှ ပေးသော ညွှန်ကြားချက်များကို လိုက်နာပါ။ CLA သည် ကျွန်ုပ်တို့၏ CLA ကို အသုံးပြုသည့် repo များအားလုံးတွင် တစ်ကြိမ်သာ လုပ်ဆောင်ရမည်ဖြစ်သည်။

ဤပရောဂျက်သည် [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) ကို လက်ခံအသုံးပြုထားပါသည်။ ပိုမိုသိရှိလိုပါက [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) ကို ကြည့်ရှုပါ သို့မဟုတ် [opencode@microsoft.com](mailto:opencode@microsoft.com) သို့ ဆက်သွယ်မေးမြန်းနိုင်ပါသည်။
