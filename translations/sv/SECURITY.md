## Säkerhet

Microsoft tar säkerheten för våra programvaruprodukter och tjänster på allvar, vilket
inkluderar alla källkodsförråd i våra GitHub-organisationer.

**Vänligen rapportera inte säkerhetsbrister via offentliga GitHub-ärenden.**

För information om säkerhetsrapportering, platser, kontaktinformation och policyer,
vänligen granska den senaste vägledningen för Microsoft-förråd på
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
