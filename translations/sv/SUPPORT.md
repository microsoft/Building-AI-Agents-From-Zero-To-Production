# Support

## Hur man rapporterar problem och får hjälp  

Detta projekt använder GitHub Issues för att spåra buggar och funktionsförfrågningar. Vänligen sök bland befintliga 
ärenden innan du rapporterar nya för att undvika dubbletter. För nya ärenden, rapportera din bugg eller 
funktionsförfrågan som ett nytt Issue.

För hjälp och frågor om att använda detta projekt, vänligen **REPOUNDERHÅLLARE: INFÖR INSTRUKTIONER HÄR 
FÖR HUR MAN KONTAKTAR REPOÄGARE ELLER GEMENSKAPEN FÖR HJÄLP. KAN VARA EN STACK OVERFLOW-TAGG ELLER ANNAN
KANAL. VAR KOMMER DU ATT HJÄLPA FOLK?**.
