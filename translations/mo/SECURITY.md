## Security

Microsoft 非常重視我們軟件產品和服務的安全性，其中包括我們 GitHub 組織中的所有源代碼庫。

**請勿通過公開的 GitHub 問題報告安全漏洞。**

有關安全報告的信息、地點、聯繫方式和政策，請查閱 Microsoft 倉庫的最新指引，網址為
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)。
