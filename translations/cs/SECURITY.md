## Bezpečnost

Microsoft bere bezpečnost našich softwarových produktů a služeb vážně, což zahrnuje všechny repozitáře zdrojového kódu v našich organizacích na GitHubu.

**Prosím, nehlaste bezpečnostní zranitelnosti prostřednictvím veřejných GitHub issues.**

Pro informace o hlášení bezpečnostních problémů, místech, kontaktech a zásadách si prosím přečtěte nejnovější pokyny pro repozitáře Microsoftu na
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
