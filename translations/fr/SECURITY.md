## Sécurité

Microsoft prend très au sérieux la sécurité de nos produits et services logiciels, ce qui inclut tous les dépôts de code source dans nos organisations GitHub.

**Veuillez ne pas signaler les vulnérabilités de sécurité via les issues publiques GitHub.**

Pour les informations sur le signalement de sécurité, les emplacements, les contacts et les politiques, veuillez consulter les dernières directives pour les dépôts Microsoft à
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
