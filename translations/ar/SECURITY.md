## الأمان

تأخذ Microsoft أمان منتجاتنا البرمجية وخدماتنا على محمل الجد، ويشمل ذلك جميع مستودعات الشيفرة المصدرية في منظمات GitHub الخاصة بنا.

**يرجى عدم الإبلاغ عن ثغرات الأمان من خلال قضايا GitHub العامة.**

لمعلومات حول الإبلاغ عن الأمان، المواقع، معلومات الاتصال، والسياسات، يرجى مراجعة أحدث الإرشادات لمستودعات Microsoft على
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
