## Sigurnost

Microsoft ozbiljno shvaća sigurnost naših softverskih proizvoda i usluga, što uključuje sve spremišta izvornog koda u našim GitHub organizacijama.

**Molimo vas da ne prijavljujete sigurnosne ranjivosti putem javnih GitHub problema.**

Za informacije o prijavi sigurnosnih problema, lokacijama, kontakt informacijama i politikama, molimo pregledajte najnovije smjernice za Microsoft spremišta na
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
