# समर्थन

## समस्या कशा नोंदवायच्या आणि मदत कशी घ्यायची  

हा प्रकल्प बग्स आणि फिचर विनंत्या ट्रॅक करण्यासाठी GitHub Issues वापरतो. डुप्लिकेट टाळण्यासाठी नवीन समस्या नोंदवण्यापूर्वी विद्यमान समस्या शोधा. नवीन समस्यांसाठी, तुमची बग किंवा फिचर विनंती नवीन Issue म्हणून नोंदवा.

या प्रकल्पाचा वापर करताना मदत आणि प्रश्नांसाठी, कृपया **रेपो मेंटेनर: येथे रेपो मालक किंवा समुदायाशी कशी संपर्क साधायची याबाबत सूचना द्या. हे Stack Overflow टॅग किंवा इतर चॅनेल असू शकते. तुम्ही लोकांना कुठे मदत कराल?**.

## मायक्रोसॉफ्ट समर्थन धोरण  

या **प्रकल्प किंवा उत्पादनासाठी** समर्थन वरील संसाधनांपुरते मर्यादित आहे.
