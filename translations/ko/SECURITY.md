## 보안

Microsoft는 소프트웨어 제품 및 서비스의 보안을 매우 중요하게 여기며, 여기에는 GitHub 조직 내 모든 소스 코드 저장소가 포함됩니다.

**보안 취약점은 공개 GitHub 이슈를 통해 보고하지 마십시오.**

보안 보고 정보, 위치, 연락처 정보 및 정책에 대해서는 Microsoft 저장소에 대한 최신 지침을
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)에서 확인해 주십시오.
