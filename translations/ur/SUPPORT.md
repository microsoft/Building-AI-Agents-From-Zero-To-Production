# سپورٹ

## مسائل درج کرنے اور مدد حاصل کرنے کا طریقہ  

یہ پروجیکٹ بگز اور فیچر ریکویسٹ کو ٹریک کرنے کے لیے GitHub Issues استعمال کرتا ہے۔ براہ کرم نئے مسائل درج کرنے سے پہلے موجودہ مسائل تلاش کریں تاکہ نقل سے بچا جا سکے۔ نئے مسائل کے لیے، اپنا بگ یا فیچر ریکویسٹ ایک نئے Issue کے طور پر درج کریں۔

اس پروجیکٹ کے استعمال کے بارے میں مدد اور سوالات کے لیے، براہ کرم **ریپو مینٹینر: یہاں ہدایات شامل کریں کہ ریپو مالکان یا کمیونٹی سے مدد کے لیے کیسے رابطہ کریں۔ یہ Stack Overflow ٹیگ یا کوئی اور چینل ہو سکتا ہے۔ آپ کہاں لوگوں کی مدد کریں گے؟**۔
