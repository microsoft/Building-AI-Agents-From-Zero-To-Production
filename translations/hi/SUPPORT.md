# समर्थन

## मुद्दे कैसे दर्ज करें और सहायता कैसे प्राप्त करें  

यह परियोजना बग और फीचर अनुरोधों को ट्रैक करने के लिए GitHub Issues का उपयोग करती है। कृपया नए मुद्दे दर्ज करने से पहले मौजूदा मुद्दों को खोजें ताकि डुप्लिकेट से बचा जा सके। नए मुद्दों के लिए, अपना बग या फीचर अनुरोध एक नए Issue के रूप में दर्ज करें।

इस परियोजना का उपयोग करने के बारे में सहायता और प्रश्नों के लिए, कृपया **रिपो मेंटेनर: यहाँ निर्देश डालें कि रिपो मालिकों या समुदाय से सहायता कैसे प्राप्त करें। यह Stack Overflow टैग या अन्य चैनल हो सकता है। आप लोगों की मदद कहाँ करेंगे?**।

## Microsoft समर्थन नीति  

इस **परियोजना या उत्पाद** के लिए समर्थन केवल ऊपर सूचीबद्ध संसाधनों तक सीमित है।
