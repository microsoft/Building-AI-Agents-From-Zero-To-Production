## பாதுகாப்பு

Microsoft எங்கள் மென்பொருள் தயாரிப்புகள் மற்றும் சேவைகளின் பாதுகாப்பை மிகுந்த முக்கியத்துவத்துடன் எடுத்துக்கொள்கிறது, இதில் எங்கள் GitHub நிறுவனங்களில் உள்ள அனைத்து மூலக் குறியீட்டு சேமிப்பகங்களும் அடங்கும்.

**பொது GitHub பிரச்சனைகள் மூலம் பாதுகாப்பு குறைபாடுகளை அறிக்கையிட வேண்டாம்.**

பாதுகாப்பு அறிக்கை தகவல்கள், இடங்கள், தொடர்பு தகவல்கள் மற்றும் கொள்கைகள் குறித்து, Microsoft சேமிப்பகங்களுக்கான சமீபத்திய வழிகாட்டுதல்களை
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md) இல் பார்வையிடவும்.
