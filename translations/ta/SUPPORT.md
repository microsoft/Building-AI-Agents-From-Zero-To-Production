# ஆதரவு

## பிரச்சனைகளை பதிவு செய்வது மற்றும் உதவி பெறுவது எப்படி  

இந்த திட்டம் பிழைகள் மற்றும் அம்ச கோரிக்கைகளை கண்காணிக்க GitHub Issues ஐ பயன்படுத்துகிறது. நகலெடுக்கப்படுவதைத் தவிர்க்க புதிய பிரச்சனைகளை பதிவு செய்வதற்கு முன் உள்ள பிரச்சனைகளைத் தேடவும். புதிய பிரச்சனைகளுக்கு, உங்கள் பிழை அல்லது அம்ச கோரிக்கையை புதிய பிரச்சனையாக பதிவு செய்யவும்.

இந்த திட்டத்தைப் பயன்படுத்துவதற்கான உதவி மற்றும் கேள்விகளுக்கு, தயவுசெய்து **ரெப்போ பராமரிப்பாளர்: உதவி பெற ரெப்போ உரிமையாளர்கள் அல்லது சமூகத்துடன் தொடர்பு கொள்ள எவ்வாறு செயல்படுவது என்பதை இங்கே உள்ளிடவும். இது ஒரு Stack Overflow குறிச்சொல் அல்லது பிற சேனல் இருக்கலாம். நீங்கள் எங்கே உதவப்போகிறீர்கள்?**.

## Microsoft ஆதரவு கொள்கை
