## Sicurezza

Microsoft prende seriamente la sicurezza dei nostri prodotti software e servizi, che
include tutti i repository di codice sorgente nelle nostre organizzazioni GitHub.

**Si prega di non segnalare vulnerabilità di sicurezza tramite issue pubblici su GitHub.**

Per informazioni sulla segnalazione di sicurezza, posizioni, contatti e politiche,
si prega di consultare le ultime linee guida per i repository Microsoft su
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
