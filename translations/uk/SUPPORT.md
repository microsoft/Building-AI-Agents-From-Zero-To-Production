# Підтримка

## Як подавати проблеми та отримувати допомогу  

Цей проєкт використовує GitHub Issues для відстеження помилок і запитів на функції. Будь ласка, пошукайте існуючі 
проблеми перед поданням нових, щоб уникнути дублікатів. Для нових проблем подавайте вашу помилку або 
запит на функцію як нову проблему.
