# Zava - Довідник працівників

## Огляд компанії
Zava — це компанія з розробки програмного забезпечення, що спеціалізується на інструментах підвищення продуктивності на базі ШІ та платформах для розробників.

---

## Організаційна структура

### Виконавча команда
- **CEO**: Підпорядковується Раді директорів
- **CTO**: Підпорядковується CEO

### Інженерні команди
- **Команда AI-інженерії**: Створює моделі ШІ/МН та інтелектуальні функції
- **Платформна команда**: Основна інфраструктура та бекенд-сервіси
- **Мобільна команда**: Додатки для iOS та Android
- **Команда DevOps**: CI/CD, інфраструктура та надійність
