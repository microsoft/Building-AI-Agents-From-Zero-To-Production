## ความปลอดภัย

Microsoft ให้ความสำคัญกับความปลอดภัยของผลิตภัณฑ์ซอฟต์แวร์และบริการของเราอย่างจริงจัง ซึ่งรวมถึงที่เก็บซอร์สโค้ดทั้งหมดในองค์กร GitHub ของเรา

**โปรดอย่ารายงานช่องโหว่ด้านความปลอดภัยผ่านปัญหา GitHub สาธารณะ**

สำหรับข้อมูลการรายงานความปลอดภัย สถานที่ ข้อมูลติดต่อ และนโยบาย โปรดตรวจสอบคำแนะนำล่าสุดสำหรับที่เก็บของ Microsoft ที่
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)
