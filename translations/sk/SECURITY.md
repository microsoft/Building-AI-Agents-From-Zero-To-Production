## Bezpečnosť

Microsoft berie bezpečnosť našich softvérových produktov a služieb vážne, čo zahŕňa všetky repozitáre zdrojového kódu v našich organizáciách na GitHube.

**Prosím, nehláste bezpečnostné zraniteľnosti prostredníctvom verejných GitHub issues.**

Pre informácie o hlásení bezpečnostných problémov, miestach, kontaktných údajoch a politikách,
si prosím prečítajte najnovšie pokyny pre repozitáre Microsoftu na
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
