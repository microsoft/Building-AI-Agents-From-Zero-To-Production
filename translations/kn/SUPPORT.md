# ಬೆಂಬಲ

## ಸಮಸ್ಯೆಗಳನ್ನು ದಾಖಲಿಸುವುದು ಮತ್ತು ಸಹಾಯ ಪಡೆಯುವುದು ಹೇಗೆ  

ಈ ಪ್ರಾಜೆಕ್ಟ್ ಬಗ್‌ಗಳು ಮತ್ತು ವೈಶಿಷ್ಟ್ಯ ವಿನಂತಿಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲು GitHub Issues ಅನ್ನು ಬಳಸುತ್ತದೆ. ನಕಲುಗಳನ್ನು ತಪ್ಪಿಸಲು ಹೊಸ ಸಮಸ್ಯೆಗಳನ್ನು ದಾಖಲಿಸುವ ಮೊದಲು ಇತ್ತೀಚಿನ ಸಮಸ್ಯೆಗಳನ್ನು ಹುಡುಕಿ. ಹೊಸ ಸಮಸ್ಯೆಗಳಿಗೆ, ನಿಮ್ಮ ಬಗ್ ಅಥವಾ ವೈಶಿಷ್ಟ್ಯ ವಿನಂತಿಯನ್ನು ಹೊಸ Issue ಆಗಿ ದಾಖಲಿಸಿ.

ಈ ಪ್ರಾಜೆಕ್ಟ್ ಬಳಕೆಯ ಬಗ್ಗೆ ಸಹಾಯ ಮತ್ತು ಪ್ರಶ್ನೆಗಳಿಗೆ, ದಯವಿಟ್ಟು **ರೆಪೊ ನಿರ್ವಹಣಾಧಿಕಾರಿ: ಇಲ್ಲಿ ರೆಪೊ ಮಾಲೀಕರ ಅಥವಾ ಸಮುದಾಯದೊಂದಿಗೆ ಸಹಾಯಕ್ಕಾಗಿ ಹೇಗೆ ಸಂಪರ್ಕಿಸಬೇಕೆಂಬ ಸೂಚನೆಗಳನ್ನು ಸೇರಿಸಿ. ಇದು ಸ್ಟಾಕ್ ಓವರ್‌ಫ್ಲೋ ಟ್ಯಾಗ್ ಅಥವಾ ಇತರ ಚಾನೆಲ್ ಆಗಿರಬಹುದು. ನೀವು ಜನರಿಗೆ ಸಹಾಯ ಮಾಡುವ ಸ್ಥಳ ಯಾವುದು?**.

## ಮೈಕ್ರೋಸಾಫ್ಟ್ ಬೆಂಬಲ ನೀತಿ  

ಈ **ಪ್ರಾಜೆಕ್ಟ್ ಅಥವಾ ಉತ್ಪನ್ನ** ಗೆ ಬೆಂಬಲವು ಮೇಲ್ಕಂಡ ಸಂಪನ್ಮೂಲಗಳಿಗೆ ಮಾತ್ರ ಸೀಮಿತವಾಗಿದೆ.
