## Безопасность

Microsoft серьезно относится к безопасности наших программных продуктов и сервисов, включая все репозитории исходного кода в наших организациях на GitHub.

**Пожалуйста, не сообщайте о уязвимостях безопасности через публичные вопросы на GitHub.**

Для получения информации о сообщении о безопасности, местах, контактной информации и политиках, пожалуйста, ознакомьтесь с последними рекомендациями для репозиториев Microsoft по адресу
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
