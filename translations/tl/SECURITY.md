## Seguridad

Seryoso ang Microsoft sa seguridad ng aming mga produktong software at serbisyo, kabilang na ang lahat ng mga source code repository sa aming mga organisasyon sa GitHub.

**Mangyaring huwag iulat ang mga kahinaan sa seguridad sa pamamagitan ng mga pampublikong isyu sa GitHub.**

Para sa impormasyon sa pag-uulat ng seguridad, mga lokasyon, impormasyon sa pakikipag-ugnayan, at mga patakaran, pakisuri ang pinakabagong gabay para sa mga repositoryo ng Microsoft sa [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
